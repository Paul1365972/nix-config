# Bidirectional Dependencies

Den's [dependency](dependencies.html) system allows you to have Hosts that provide configuration to all their users and users that provide configuration to the hosts they live on.

See [some](templates/examples/modules/_example/ci/host-configures-users.nix) [examples](templates/examples/modules/_example/ci/user-configures-hosts.nix) of [this](templates/examples/modules/_example/ci/one-os-package-per-user.nix).
